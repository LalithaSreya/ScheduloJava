# ScheduloJava

ScheduloJava is a Java-based task scheduling and management system designed to help users organize tasks, track schedules, and manage activities efficiently through a simple console-based interface.

## Overview

This project demonstrates the implementation of core Java concepts such as object-oriented programming, file organization, and modular design. It provides a structured approach to handling tasks, managing schedules, and maintaining basic statistics.

## Features

* Task creation, update, and deletion
* Task scheduling and organization
* Calendar-based task management
* Basic statistics tracking

## Technologies Used

* Java (Core Java, OOP concepts)

## Project Structure

* `Main.java` – Entry point of the application
* `Task.java` – Represents individual tasks
* `TaskManager.java` – Handles task operations
* `CalendarManager.java` – Manages scheduling and dates
* `StatsManager.java` – Tracks and displays statistics

## How to Run

1. Clone the repository:
   git clone https://github.com/your-username/ScheduloJava.git

2. Navigate to the project directory:
   cd ScheduloJava

3. Compile the program:
   javac Main.java

4. Run the program:
   java Main

## Output

The application runs in the console and allows users to interact with the system through text-based input and output.
